# Manipulação de Dados

Esta pasta contém práticas e exemplos de manipulação de dados com Pandas, CSV e Excel.

## Datasets

- **IMDB**: Dados de filmes e elenco
- **Excel**: Exemplos de leitura e escrita
- **empresa.db**: Banco de dados SQLite de exemplo

## Scripts

- `pratica_panda.py`: Exercícios com Pandas
- `pratica_filtrodedados.py`: Filtragem de dados
- `xls.py`: Manipulação de Excel
- `create_sample_excel.py`: Criação de arquivos Excel
